## 这是什么？

最近海外很火的「多巴胺解压网站」，我们做了**中文版独立站**：[foodnevercomes.cn](https://foodnevercomes.cn)

它不是外卖 App，也不是骗你下单的平台。  
这是一场**零消费的虚拟点单体验**：

- 🍜 浏览 8 国美食，随便加购  
- 💳 一键「付款」——假银行卡，永不扣款  
- 🛵 实时追踪骑手（乌龟、火箭、独角兽都可能来送）  
- 🎉 「已送达！」——饭没到，快乐到了

## 怎么玩？

1. 打开 **foodnevercomes.cn**  
2. 选菜 → 加购 → 结账  
3. 看骑手配送动画  
4. 点击 **「分享晒单」**，生成带二维码的图片，发给朋友扫码一起玩
